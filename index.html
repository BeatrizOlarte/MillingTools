KG4CUT - Knowledge Graph for Cutting Tools

Description

This ontology describes and models solid end milling tools, with the aim of creating interoperability across different vendors. Our ontology models the solid end mills, linking geometry, cutting parameters and milling tasks.

This ontology allows mapping different cutting tool manufacturers information for solid end mills, enabling interoperability. An application as cutting tool recommendation engine is being developed.

Fig. 1: Visual ontology summary

Our ontology reuses InVor ontology, componency from ODP and foaf vacbulary (see References).

Fig. 2: Imported ontologies in KG4CUT

Major Classes

Some of the major classes defined in the ontology are Milling Task and ISO Materials. Milling Tasks define all the major and more known and use milling operations and links some of them to specific geometries. ISO materials is necessary as a particular way to define the agree taxonomy of materials in chip removal processes.
